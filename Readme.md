# asynCmd

## Readme

### How to Use asynCmd?

The command `asyncmd` will show you a little Example as Help.

After the Command `asyncmd` you can give Parameter like `asyncmd "apt-get install htop" "bash"`, this will install htop and open a new Bash on Linux;

---

WINDOWS EXAMPLE:

`asyncmd cmd pause`
